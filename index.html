Обо мне

Я опытный QA Engineer с техническим образованием и практическим опытом работы в области тестирования программного обеспечения. Моя специализация — обеспечение качества с помощью ручного тестирования, анализа требований, разработки тест-кейсов и тест-дизайна. Я имею опыт работы с клиент-серверной архитектурой и знаком с основами автоматизированного тестирования. Мои сильные стороны включают внимание к деталям, способность выявлять даже неочевидные ошибки и умение работать в слаженной команде для достижения общей цели. Открыт к освоению новых инструментов и подходов в области тестирования и разработки.

Навыки

  • Тестирование: опыт ручного тестирования веб-приложений и API, составление баг-репортов, создание и выполнение тест-кейсов, работа с чек-листами.
  • Инструменты тестирования: опыт работы с Postman для тестирования API, использование DevTools для анализа работы веб-приложений, практическое использование TMS (Qase).
  • Тест-дизайн: знание техник тест-дизайна.
  • API и Web-технологии: знание принципов клиент-серверной архитектуры, работы с REST API, JSON, а также опыт работы с Postman для автоматизации запросов и тестирования
  • Программирование и Скрипты: основы JavaScript для написания автотестов, опыт работы с Python для автоматизации задач.
  • Базы данных: работа с SQLite, базовые навыки работы с SQL-запросами
  • Версионный контроль: опыт работы с Git для управления версиями кода, командной работы и интеграции с CI/CD процессами
  • Инструменты для анализа трафика: опыт работы с сниффером Charles для анализа HTTP/HTTPS-запросов
  • Командная работа и взаимодействие: опыт работы в команде с разработчиками и другими заинтересованными сторонами для решения задач, связанных с качеством продукта
